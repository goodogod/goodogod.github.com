<h2>{{ page.title }}</h2>
Homebrew 是 Mac OS X 下的套件套件管理程式，它提供類似 apt-get 的命令行工具。

安裝 homebrew：
>ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go/install)" 

安裝 node：
>brew install node

安裝 npm (Node Packaged Modules)：
>brew install npm

安裝 Express：
>npm install express

安裝 MongoDB：
>npm install mongodb
